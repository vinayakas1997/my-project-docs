Year2024 — Present

Overview

This project involved modernizing factory communication at Somic Ishikawa by migrating legacy proprietary protocols to a standardized OPC UA-based architecture. The goal was to enable real-time, reliable data exchange between PLCs and higher-level systems for monitoring and analytics.

A custom FINS protocol handler in Python was developed to bridge Omron PLCs into the OPC UA server, enabling seamless data access without replacing existing hardware.
