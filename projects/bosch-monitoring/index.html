Bosch Soldering Process Monitor

Analyzing heat-induced failures in selective soldering with 3 POC monitoring models including computer vision

CompanyBosch Automotive Electronics
RoleEngineering Intern
YearJan — Mar 2019
LocationBangalore, India
CV

Overview

During an internship at Bosch Automotive Electronics, analyzed heat-induced failures in the selective soldering process that reduced filter assembly reliability and increased chip wastage. Built 3 POC monitoring models to identify root causes and define optimal operating windows.

Monitoring Capabilities

From heat-failure analysis through POC prototypes to data-driven operating windows.

Heat-failure root-cause analysis

Studied heat-induced filter reliability loss in selective soldering — linking thermal stress patterns to assembly failures and chip wastage.

3 POC monitoring models

Proposed and built three proof-of-concept approaches to compare monitoring strategies before committing to production instrumentation.

Mini-camera computer vision setup

Deployed a mini-camera CV pipeline for visual process monitoring — capturing solder joint quality signals without disrupting the line.

Optimal operating window identification

Combined POC findings to define data-driven operating windows and maintenance timing recommendations for production teams.

Challenges Tackled

Process reliability issues that motivated the monitoring POC work.

Heat-induced filter reliability loss

Thermal stress during selective soldering degraded filter assembly quality — requiring systematic failure analysis before monitoring could be designed.

Chip wastage in selective soldering

Material loss from process variability demanded measurable signals — not just post-hoc inspection — to catch drift before scrap accumulated.

Maintenance timing without production data

Teams lacked objective usage and condition data to schedule maintenance — POC models explored time-based and visual proxies for equipment health.

Key Contributions

  • Analyzed heat-induced failures causing reduced filter reliability and chip wastage
  • Proposed and built 3 POC monitoring models, including a mini-camera–based setup and time-based usage tracking
  • Enabled data-driven identification of optimal operating windows and maintenance timing
  • Supported reduced material wastage and improved production efficiency through findings
