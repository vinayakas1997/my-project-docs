# Glossary — Quick Reference

| Term | Full Form | Simple Explanation |
|---|---|---|
| **AST** | Abstract Syntax Tree | Tree representation of code structure — lets you analyze SQL logically, not as raw text |
| **CTE** | Common Table Expression | Temporary named result set in SQL (`WITH name AS (...) SELECT ...`). Useful for breaking complex queries but can hide mutations |
| **DGX Spark** | — | NVIDIA's edge AI appliance — runs LLMs on-premise without cloud APIs |
| **FDE** | Forward Deployed Engineer | Engineer who embeds with enterprise customers to deploy AI solutions in production |
| **FINS** | Factory Interface Network Service | Omron's proprietary protocol for PLC communication over Ethernet |
| **HITL** | Human-in-the-Loop | System where a human can review, override, or validate AI outputs before they're final |
| **Hindsight** | — | A structured memory/knowledge graph platform for long-term AI context retention |
| **Levenshtein distance** | — | Minimum character operations (insert/delete/substitute) to turn one string into another. E.g., `"cat"` → `"cats"` = 1 (insert s), `"$450.00"` → `"$450.50"` = 1 (substitute 0→5). |
| **LLM** | Large Language Model | AI model trained on vast text data (e.g., GPT-4, Qwen) that generates human-like text |
| **MinerU** | — | OCR engine specialized for layout-aware document parsing (tables, rotated text, diagrams) |
| **OCR** | Optical Character Recognition | Converting images of text into machine-readable text |
| **OPC UA** | Open Platform Communications Unified Architecture | Standard protocol for industrial machine-to-machine communication |
| **PLC** | Programmable Logic Controller | Industrial computer that controls manufacturing equipment |
| **RAG** | Retrieval-Augmented Generation | AI pattern: retrieve relevant documents first, then generate an answer grounded in those documents |
| **sqlglot** | — | Python library that parses SQL into an AST, supports multiple SQL dialects |
| **VLM** | Vision Language Model | AI model that understands both images and text (e.g., GLM-4.6V Flash) |
